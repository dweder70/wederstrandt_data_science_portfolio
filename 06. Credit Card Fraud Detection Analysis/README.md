# Credit Card Fraud Detection Analysis

## Overview

This project builds and evaluates a machine learning workflow for credit card fraud detection, with emphasis on imbalanced classification.

## Objective

Detect fraudulent transactions while balancing false positives and false negatives in a high-cost decision context.

## Files

- credit_card_fraud_detection_analysis.ipynb: End-to-end notebook for data preparation, model training, and evaluation.
- README.md: Project summary and reproducibility guidance.

## Methods

- Exploratory analysis of target class imbalance.
- Data cleaning and feature preparation.
- Supervised model training and comparison.
- Evaluation using precision, recall, F1, and ROC-AUC.

## Tools

- Python
- Jupyter Notebook
- pandas, numpy, scikit-learn, matplotlib, seaborn

## How To Reproduce

1. Open the notebook in Jupyter or VS Code.
2. Install required Python packages.
3. Run all cells in sequence.

## Outcome

The notebook demonstrates practical fraud modeling decisions for imbalanced data and focuses on metrics that matter for real-world risk operations.
